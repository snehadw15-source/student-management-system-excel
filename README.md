# 🎓 Student Management System – Excel Project

An Excel-based Student Management System that tracks academic records, 
automates grading and pass/fail evaluation, monitors attendance and fee 
status, and presents performance through an interactive KPI dashboard.

## 🚀 Features
- Auto-calculated Total, Percentage, and Grade (VLOOKUP)
- Pass/Fail evaluation using nested IF + AND
- Data Validation (dropdowns, marks restricted 0-100)
- Conditional Formatting (Pass/Fail, low attendance, pending fees)
- KPI Dashboard: Total Students, Class Average, Top Scorer, Pass/Fail counts
- COUNTIF, COUNTIFS, AVERAGEIF, INDEX/MATCH
- Pivot Table, Pivot Chart & Slicers (Class, Section, Gender, Fee Status)
